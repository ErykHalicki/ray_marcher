# Music Marcher - Procedurally Generated Audio Visualizer

Project developed for StormHacks 2025 at Simon Fraser University.

Utilizes glsl shaders, SDL3, and FFTW3 to generate a ray marched, procedurally generated scene that reacts to the music playing on your computer in real time.
